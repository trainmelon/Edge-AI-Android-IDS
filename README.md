# Edge-AI-Android-IDS (基于端侧 AI 的实时网络入侵检测系统)

## 📌 项目简介
本项目是一个基于**端侧计算 (Edge AI)** 的移动端网络入侵检测系统 (IDS)。
项目采用**“云端训练量化 + 移动端本地推理”**的闭环架构：在算法侧，基于 NSL-KDD 数据集训练多分类深度学习模型，并运用 Int8 训练后量化 (PTQ) 技术突破移动端算力瓶颈；在工程侧，使用 Kotlin 独立开发 Android 原生 App，在**纯离线环境**下实现极低延迟的实时端侧推理与威胁告警。

## 🚀 核心特性 (Core Features)
* **端侧脱机推理 (Edge Inference)：** 彻底摆脱对云端服务器的依赖，保护用户数据隐私，实现“断网可测”。
* **五大攻击类型细粒度分类：** 模型不仅能识别异常，还能精准划分网络攻击的具体细分类型（如 DoS, Probe, R2L, U2R 等）。
* **轻量化部署 (Model Quantization)：** 运用 TFLite 与 Int8 量化技术，将模型体积与推理能耗压缩至极低，完美适配算力受限的移动设备。
* **物理网络探针与能耗调度：** 结合 Android 原生 API 开发了真实网络环境探针，并实现了动态能耗调度与告警闭环。

## 📁 项目架构与源码说明
* `model_training/`：算法侧源码。包含基于 Python 的数据预处理、多分类 MLP 模型训练、多种经典算法（LR/DT/RF/MLP）对比验证，以及 TFLite Int8 量化导出脚本。
* `android_app/`：工程侧源码。基于 Kotlin 开发的 Android 原生 App，包含 UI 逻辑、TFLite 推理引擎调用及网络监控模块（剔除了冗余的 build 编译缓存）。
* `assets/`：包含模型训练过程中的各类对比曲线图与实验结果验证截图。
